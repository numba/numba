Add Windows on ARM64 to Tier 1.5 support policy
-----------------------------------------------

The :ref:`Support Tiers <support_tiers>` documentation now lists ``win-arm64``
(Windows on ``aarch64``/``arm64``) as a Tier 1.5 platform.  Wheel packages are
initially provided for Python 3.14 only; support for additional Python versions
may be added in the future.
